# Docker PHP boilerplate
Simple Docker boilerplate for Symfony projects

# Status
Work In Progress!

## Features
- PHP with built in Httpd server
- Maria DB
- Symfony CMD
- Composer
- PHPUnit
- Phan

## Requirements
- Docker 18.09.7 or newer
- Docker compose 1.27.4 or newer

## Usage
- `docker-compose up`
